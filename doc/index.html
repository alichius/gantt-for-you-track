@fschopp/gantt-for-you-track

Gantt charts for YouTrack. Computes a project plan for the unresolved issues in a YouTrack saved search and visualizes it as Gantt chart, together with the past activity log.

Status

Overview
  • Try it here and now.
  • Based on:
    • fschopp/project-planning-ui-for-you-track for its user-interface framework and for configuring the connection to YouTrack.
    • fschopp/project-planning-for-you-track for the communication with YouTrack.
    • fschopp/project-planning-js for the list-scheduling algorithm used for scheduling the unresolved issues.
    • adamhaile/S.js and adamhaile/Surplus for reactive programming of the user interface.
    • dhtmlxGantt for the Gantt widget.
  • Written in TypeScript.
  • Partial API documentation available. Generated by TypeDoc.
  • Test coverage is currently incomplete. However, the underlying algorithm packages fschopp/project-planning-js and fschopp/project-planning-for-you-track have complete test coverage.
